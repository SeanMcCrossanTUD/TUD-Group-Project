Step-by-Step Guide

1. Upload Your Data

Upon logging in, you will seamlessly transition to this page where you can upload your data to start the journey. Supported file formats include CSV and Excel, with a maximum size limit of 20MB. You can use the 'Choose' button or drag-and-drop your file, then click 'Upload' button to proceed. Clcik on "Next" button to navigate to second page.



2. Preview Uploaded Data

On this page, you can view the first 200 rows of your uploaded data. Enhance your viewing experience with various features including:

  • Expand columns to view full content.
  • Auto-size individual columns or all columns using 'Autosize This Column' and 'Autosize All Columns' options.
  • Reset columns after applying filters.
  • Adjust column positions as needed.
  • Pin columns to the left or right.
  • Sort column contents in ascending or descending order.
  • Utilize the 'Filters' option to refine the contents of each column, with the ability to select, deselect, and use a search bar.
  • Unlock insights with the Pivot table feature, dynamically reorganizing and summarizing data for a customized and insightful view.
  • Optimize organization and analysis by implementing Row Groupings feature.
  • After implementing row groupings, apply various value aggregations such as average, count, first, last, maximum, minimum, and summation.
  • Further features post row groupings include 'Un-Group All', 'Group by Group', 'Expand All Row Groups' and 'Collapse All Row Groups' to facilitate group management.



3. Data Insights Dashboard - Navigating Data Quality, Profiling, and Exploration

The Data Polish Dashboard offers a user-friendly interface designed for effectively managing and exploring data, organized into three distinct tabs.


Data Profile

Designed to offer a comprehensive overview, the Data Profile tab aids in understanding the fundamental structure of your data like- record count, field count, cardinality of text fields and data type distribution.




Data Quality

Designed to enhance your awareness of data quality, the Data Quality tab is a powerful tool that identifies issues and provides guidance for effective data cleaning. Its key features include- Duplicate Record Identification, Data Quality Score (Completeness, Uniqueness, Consistency and Readability), Missing Values Chart.




Data Exploration

Unlock the potential of your data with the interactive and enlightening Data Exploration tab on your dashboard. Dive into the depths of your dataset, gaining valuable insights and visualizing the intricate relationships and complexities within it. Its key features are- Numerical Data Distribution, Correlation Discovery, Outlier Scatter Plot Visualization, Bubble Chart for Categorical Data.




4. Perform Data Cleaning Actions

Data Cleaning page has two adjustable panels. The bottom panel provides you with the Data Preview section from the second step in the application. The top panel includes all the features related to data cleaning functions.

You can use the top buttons to perform some data cleaning actions- Keep All, Remove All, Reset Rules, Dataset Actions (Remove Duplicate), Save Rules, Download Cleaning Rules, Import Cleaning Rules. Upon clicking on "Cleaning Options", a modal will pop-up with more cleaning features divded in tabs. You can choose your particular cleaning setup and save the cleaning rules. Finally, proceed to next page to get your refined data.




5. Download Cleaned Data

After configuring your data cleaning preferences in the preceding steps, download your refined data. Simply choose the desired file format, either CSV or Excel, from the drop-down menu. Complete the process by clicking the 'Export' button, making it hassle-free to download the cleaned data tailored to your specifications.
